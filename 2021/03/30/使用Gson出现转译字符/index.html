使用Gson出现转译字符

json 2021/03/30

1.出现转译字符

String s = new Gson().toJson(encodedText);出现转译字符

原因：Gson会把html标签，转换为Unicode转义字符。

正确的使用方法是:Gson gson = new GsonBuilder().disableHtmlEscaping().create();
注意：用AES加密后的字符串转义后会变
